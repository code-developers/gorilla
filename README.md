# gorilla

A Programming Language Built Using Crystal-Lang

## Installation

```
$ git clone https://github.com/krishpranav/gorilla
$ cd gorilla
$ crystal build src/gorilla.cr --release --stats -o bin/gorilla
```

## Supported Operating Systems:
- macOS
- Linux
